COMP 4350 Group1: Twindr
========================

Setup Instructions:

In order to compile and run the app you must install one of the build dependencies called Fabric.
1.  Download the required binary as provided at:
        https://dl.dropboxusercontent.com/u/26388437/Fabric.app.zip
2.  Extract and run Fabric.app
3.  After the application starts you are done
    You should now be able to compile and run the app.
    Note:
    You may disregard the login prompt for Fabric, if it's running the build will work.
